• 프로젝트 소유자 또는 초대 코드를 가진 게스트 조회 가능

POST /api/v1/feedback
피드백 등록 (게스트용)

게스트(프로젝트 의뢰인)가 디자인에 대한 피드백을 등록합니다.
초대 코드를 통해 권한을 확인합니다.

PUT /api/v1/feedback
피드백 수정 (게스트용)

게스트(프로젝트 의뢰인)가 자신이 작성한 피드백의 내용을 수정합니다.
초대 코드를 통해 권한을 확인합니다.

GET /api/v1/feedback
피드백 목록 조회 (리비전 단위)

게스트(프로젝트 의뢰인) 또는 프로젝트 소유자가 특정 리비전의 피드백 목록을 조회합니다.
로그인된 경우 프로젝트 소유자 권한을 확인하고, 그렇지 않은 경우 초대 코드를 통해 게스트 권한을 확인합니다.
